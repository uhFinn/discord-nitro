# discordnitro
**_This NPM package generates unchecked discord nitro codes for you!_**

Context:
```js
const nitro = require('discordnitro')

console.log(nitro(5))
```
Output:
> [5 Unchecked Nitro Codes]


__Another Example__
```js
const nitro = require('discordnitro')

console.log(nitro(1000000))
```
Output:
> [1,000,000 Unchecked Nitro Codes]
